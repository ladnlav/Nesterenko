# О ветках в Git

## Как создать ветку
1. Проверить, какие ветки существуют `git branch --all`
2. Создать ветку `git branch *имя ветки*`
3. Проверить, её появление `git branch --all`

## Как переключиться на другую ветку
Команда для переключения на другую ветку `git checkout *имя ветки*`

## Первый push в новорожденной ветке
Для того, чтоб сделать первый *push* в новой ветке, вводим следующую команду: `git push --set-upstream origin *имя ветки*`
